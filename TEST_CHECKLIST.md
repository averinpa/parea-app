## 10. AI MATCHING

### 10.1 Companion matching (Vibe Check)
- [ ] Correct API key → запрос уходит в claude-haiku, возвращается JSON с scores
- [ ] Scores сортируют companion cards (высший вверху)
- [ ] No API key → dummy score 75 для всех
- [ ] JSON parse error → fallback массив, без краша

### 10.2 Dealbreaker hard filters
- [ ] `no_smoking`: курильщики полностью исключаются из результатов
- [ ] `sober_only`: heavy drinkers исключаются
- [ ] `no_drugs`: пользователи с drugs исключаются
- [ ] `pets_allergy`: pet owners исключаются
- [ ] Остальные dealbreakers: `no_loud`, `no_kids` влияют на score

### 10.3 Scoring логика
- [ ] Совпадение интересов + музыки даёт 40% веса
- [ ] Lifestyle совместимость: 25%
- [ ] Language overlap: 20%
- [ ] Age proximity: 15%
- [ ] Итог: number от 0 до 100

### 10.4 Photo safety check
- [ ] Безопасное фото → verified
- [ ] NSFW фото → rejected + alert
- [ ] Фото с несовершеннолетними → rejected
- [ ] Network error во время проверки → фото допускается (fail-open)

### 10.5 Event fit scoring
- [ ] Category match: +35 баллов
- [ ] Title keyword match: +15 баллов
- [ ] Group size preference match: +20 баллов
- [ ] Минимальный base score: 30

- [ ] Незавершённый онбординг (нет фото) → нельзя нажать Next на шаге 2

- [ ] Profile color генерируется при создании и не изменяется

### 11.3 События
- [ ] Прошедшее событие автоматически скрывается (time parsing)

- [ ] Событие без времени: дедлайн по умолчанию 23:59:59

### 11.7 Realtime subscriptions
