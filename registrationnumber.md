Роль: Ты — Senior React Native / Flutter разработчик (выбери свой стек).
Задача: Создать экран авторизации по номеру телефона для приложения по поиску компаньонов на мероприятия.

Функциональные требования:

Поле ввода телефона: * Реализуй маску ввода (например, +7 (XXX) XXX-XX-XX).

Используй числовую клавиатуру (phone-pad).

Кнопка «Получить код» должна быть неактивна, пока номер не введен полностью.

Тестовый OTP-режим:

После нажатия «Получить код» покажи поле для ввода 4-значного кода.

Реализуй заглушку: если введен код 1234, авторизация успешна. Если другой — покажи ошибку «Неверный код».

Добавь визуальный таймер обратного отсчета «Отправить код повторно через 59 сек».

Юридический блок (Terms & Privacy):

В нижней части экрана размести текст: «Нажимая кнопку "Продолжать", вы принимаете [Условия использования] и [Политику конфиденциальности]».

Сделай названия документов кликабельными ссылками (используй TouchableOpacity или RichText с синим цветом).

При клике должна вызываться функция-заглушка console.log или открытие Alert, чтобы я мог позже вставить ссылки.

UI/UX:

Стиль: современный, минималистичный, чистый.

Обработка состояний: loading (загрузка при проверке кода) и error (красный текст ошибки).

Автофокус: при переходе к вводу OTP фокус должен автоматически вставать на первую ячейку кода.

Технические детали:

Используй чистый код с разделением логики и верстки.

Если используешь React Native, используй Lucide-react для иконок (если нужно) и StyleSheet для стилей.
